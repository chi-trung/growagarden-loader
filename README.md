# 🌱 Grow a Garden — Script Loader

## Usage
Paste this into your executor:

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/chi-trung/growagarden-loader/main/source.lua"))()
```

## Features
- 🌾 Auto Farm (Buy, Plant, Harvest, Sell)
- 🛒 Seed Rarity Filter + Specific Seeds
- 🔧 Auto Buy/Equip Gear
- 💧 Auto Water, Sprinkler, Gnome
- 🌈 Rainbow/Gold Seed Pack Collect
- 🏴‍☠️ Auto Steal
- 🐕 Wild Pet Tame
- 🥚 Auto Egg/Crate
- 📬 Mailbox Claim
- 📜 History Log
- 🔑 Key System
- 🛡️ Anti-AFK
